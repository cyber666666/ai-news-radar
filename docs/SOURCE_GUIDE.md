# 网页来源说明

这份文档帮助普通读者理解网页卡片上的来源名称。来源标签表示“雷达从哪里获得这条线索”，不一定表示最终事实的原创发布者。

网页中的来源大致分为四层：

1. **官方一手来源**：AI 公司、研究机构和开发平台自己的 RSS、新闻页或更新日志。
2. **媒体与 Newsletter**：记者、编辑或作者对事件的报道与整理。
3. **聚合与社区信号**：用于发现热点，但需要继续查看原文和讨论上下文。
4. **私人高级来源**：需要维护者自己的 API Key，通过 GitHub Secrets 或环境变量启用。

## AI HOT 的子来源标签

- **X**：原始线索来自 X 帖子，通常是作者观点、产品预告或即时消息。
- **公众号**：原始线索来自微信公众号文章。
- **HN**：原始线索来自 Hacker News 讨论。
- **RSS**：原始线索来自公开 RSS 或 Atom Feed。

AI HOT 是聚合和编辑入口，不能因为卡片显示 `AI HOT` 就把内容理解为 AI HOT 原创。

## 一个简单判断方法

阅读卡片时，可以按以下顺序判断：

1. 先看它是不是官方一手来源。
2. 如果是媒体或聚合来源，继续打开原文并寻找最初出处。
3. 如果来自 X、公众号或 HN，把它先当作观点或线索。
4. 多个独立来源共同报道时，可信度通常高于单一转载，但仍需确认它们是否引用了同一个原始消息。
